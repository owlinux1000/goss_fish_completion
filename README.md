# goss_fish_completion

fish completion configuration for goss

## How to use

Please ```git clone``` this repo then, move ```goss.fish``` to ```~/.config/fish/completions```. 
Also, If you are fisherman user, you can install via fisher.
```
$ fisher add owlinux1000/goss_fish_completion
```
